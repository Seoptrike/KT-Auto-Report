사용 설명서

Acen 정산 메일을 받은 후:
1. Acen 정산 파일 메일을 받으면 엑셀 파일을 업로드 합니다.
2. 세금계산서 역발행한 날짜를 입력합니다.
3. ACEN 실행 버튼을 눌러 해당 월의 새로운 'Acen 매출결의서' 파일을 생성합니다.

AICC 정산이 완료되면 :
1. 비즈포탈 접속 후 정산 메뉴 - 정산결과조회 눌러 좌측 로그인 정보 밑 정산결과를 조회합니다.
2. 정산사업자 넷컴솔루션 검색합니다.
3. 채널정산결과 목록을 하나씩 차례대로 눌러 아래 채널정산결과상세 목록을 다운로드 받습니다.
4. 모든 엑셀을 다운 받은 후 일괄 업로드 합니다.
5. 세금계산서 발행한 날짜를 입력합니다.
6. AICC 실행 버튼을 눌러 최종 '업무실적' 파일과 'AICC 매출결의서' 파일을 생성합니다.

주의: 반드시 ACEN을 먼저 실행하여 월 파일을 생성해야 AICC 작업이 가능합니다.
ACEN 실행

월초 1회, 새 파일 생성

AICC 실행

AICC 정산 후, 최종 업무실적 파일 생성
